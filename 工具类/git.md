# git常用指令如下

```shell
- git checkout [branch]   // 切换分支
- git branch [branch]  // 创建分支
- git branch -d [branch]   // 删除本地分支
- git push origin :[branch] // 删除远程分支（注意origin后空格）
- git pull
- git fetch
- git merge
- git add -A
- git commit -m"commit信息"
- git push origin [branch] 推到远程分支

```
